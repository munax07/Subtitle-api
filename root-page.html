API Endpoints

GET /subtitle?action=search Search subtitles ▾
Search OpenSubtitles by movie/show name. Supports lang filter (ISO 639-1) and pagination.
/subtitle?action=search&q=Inception&lang=en&page=1copy
GET /subtitle?action=download Download subtitle file ▾
Download subtitle by ID. Returns binary file with Content-Disposition header. Optional filename param.
/subtitle?action=download&id=3924710copy
GET /languages?q={query} Available languages ▾
Returns a sorted list of all available language codes for a movie or show.
/languages?q=Interstellarcopy
GET /stats Server statistics ▾
Returns uptime, memory usage (rss, heapUsed, heapTotal), and cache hit/miss statistics.
/statscopy
GET /health Health probe ▾
Koyeb health check endpoint. Returns status + platform + uptime. Includes X-Koyeb-Health header.
/healthcopy
GET /dashboard This interface ▾
This live monitoring dashboard. Served from dashboard.html in the project root.
/dashboardcopy
